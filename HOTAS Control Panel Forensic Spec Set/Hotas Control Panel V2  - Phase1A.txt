# Phase 1A — Thrustmaster T-Flight HOTAS One Driver and Device Setup Guidance

## Goal

Add explicit setup guidance for the known physical controller: **Thrustmaster T-Flight HOTAS One** / **Thrustmaster T.Flight Hotas One**.

This phase should not blindly install drivers. Instead, it should make the app and documentation smart enough to guide the user through installing the official Thrustmaster driver/software safely, then verify whether the controller is visible to Windows.

The driver goblin rule is simple: official source only. No shady “totally-real-driver-download-trust-me-bro.exe” nonsense.

## Official Driver Source Policy

Codex must use the official Thrustmaster support site as the source for the HOTAS driver/software.

Current official source notes:

- Product support page: `T.Flight Hotas One`
- Driver package currently resembles: `Drivers - Package 2025_TFHT_5 + Firmware`
- System requirement currently listed: Windows 10 / Windows 11

Because driver packages change over time, Codex should re-check the official Thrustmaster support page at implementation time.

## Important Driver Distinction

Thrustmaster software and vJoy are separate things:

- **Thrustmaster driver/software** is for recognizing and configuring the physical T-Flight HOTAS One on Windows.
- **vJoy or another virtual output backend** creates the virtual joystick output device that HOTAS Control Panel writes to.

Both may be needed for full live runtime.

## What Codex May Do

Codex may:

- add a Runtime Setup article explaining how to install the official Thrustmaster driver;
- add an `Open Official Thrustmaster Support Page` button;
- add a `Run HOTAS Detection` / `Run Preflight Check` button;
- detect whether a Thrustmaster/T.Flight/HOTAS One device appears in Windows device enumeration;
- detect whether the Thrustmaster Control Panel/app appears installed if there is a safe way to check;
- show setup status such as:
  - `Thrustmaster Driver Unknown`
  - `Thrustmaster Driver Missing`
  - `HOTAS Not Connected`
  - `T-Flight HOTAS One Detected`
  - `Input Ready`
  - `vJoy Missing`
  - `Simulation Mode Active`
  - `Full Live Runtime Ready`

Codex must not:

- silently install Thrustmaster drivers;
- silently install vJoy;
- download drivers from non-official mirrors;
- bundle driver installers without permission/licensing review;
- claim the HOTAS is working unless Windows/device enumeration actually sees it;
- claim vJoy output is working unless the virtual output backend is detected and verified.
