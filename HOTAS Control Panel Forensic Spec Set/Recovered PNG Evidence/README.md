# Recovered PNG Evidence

This folder contains the recovered PNG assets sorted by visible HOTAS Control Panel tab/page first, then by version or view.

Naming convention:
- `legacy-...` means the screenshot visibly shows `V2 draft`, `Save V2 Draft`, `Safe parallel Qt frontend`, or the older draft-style V2 shell. This is a naming label for rebuild evidence; it does not necessarily mean the original pre-Qt app.
- `v2-...` means the screenshot is the later/polished V2 UI or a V2 product asset and does not show the draft shell indicators.
